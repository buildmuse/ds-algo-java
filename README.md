# DS Algo Java

A Java project for learning and implementing Data Structures and Algorithms.

## Project Structure

- `src/main/java/com/preethi/ds_algo/` - Main source code
- `src/test/java/com/preethi/ds_algo/` - Test files

## Building the Project

```bash
mvn clean compile
```

## Running Tests

```bash
mvn test
```

## Running the Application

```bash
mvn clean compile exec:java -Dexec.mainClass="com.preethi.ds_algo.Main"
```

## Java Version

This project uses **Java 21 LTS**.
